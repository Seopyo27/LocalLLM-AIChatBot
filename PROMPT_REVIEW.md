\# 프롬프트 검토 기준



AI가 작성한 코드 요청 프롬프트를 아래 기준으로 검토하고 추가할 내용 추가해줘



\## 검토 항목



1\. 한 번에 하나의 기능만 요청하는가?

2\. 첫 번째 기능이 실행해서 확인 가능한가?

3\. C++ 초보자가 이해할 수 있는 수준인가?

4\. 코드 설명을 요구하고 있는가?

5\. 실행 방법을 요구하고 있는가?

6\. 검증 방법을 요구하고 있는가?

7\. 발생 가능한 오류나 주의점을 요구하고 있는가?

8\. 개선 방향을 질문하고 있는가?



\##  프롬프트 처음 부분 추가 내용

1. main 최신화 후 작업에 따른 브렌치 생성 후 체크아웃



\##  프롬프트 마지막 부분 추가 내용

1. 작업이 완료되면 테스트 실행
2. 커밋 메세지 간단하게 작성 후 커밋
3. &#x20;작업 후 추가, 삭제, 수정한 부분을 파트별로 나눠서 상세하게 설명



\## AI에게 요청할 검토 프롬프트



아래 프롬프트 초안을 위 기준에 따라 검토해줘.



검토할 때는 다음 형식으로 답변해줘.



1\. 항목별 평가

2\. 부족한 점

3\. 수정 방향

4\. 개선된 최종 프롬프트
